Controle qualite avant export

Les outils PDF doivent rester locaux par defaut: verifier le fichier, choisir le bon flux, puis telecharger seulement apres avoir relu le resultat. Les recettes, briefs et exports de workflow doivent rester des metadonnees, jamais le contenu brut du document.

Methode: commencez par compresser, fusionner, convertir, signer ou rediger selon le besoin; gardez une copie source; controlez les pages, la confidentialite, la lisibilite et le format avant partage.
