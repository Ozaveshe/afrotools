Questions frequentes

Dois-je choisir une ligne dediee ?
Une ligne dediee peut aider lorsque la connexion porte les ventes, la caisse, la VoIP ou le support client. Pour un petit bureau, une bonne fibre partagee avec une solution de secours mobile peut suffire.
Les couts affiches sont-ils des devis operateur ?
Non. L outil utilise les donnees telecom deja presentes dans AfroTools pour comparer des ordres de grandeur. Le devis final doit venir du fournisseur.
Pourquoi garder une connexion de secours ?
Dans beaucoup de villes africaines, une panne fibre, energie ou quartier peut interrompre les ventes. Une route LTE, 5G ou mobile data limite le risque sans dupliquer toute l infrastructure.
