    Methodology

    Le numéro est nettoye en chiffres uniquement. Si un indicatif pays est choisi, le zero local de tete est retire puis l'indicatif est ajoute. Le message utilise encodeURIComponent et devient le paramètre text du lien.

    Questions fréquentes

    Dois-je garder le signe plus dans le lien ?

    Non. Le format wa.me utilise le numéro international sans le signe plus, sans espaces et sans tirets.

    Le lien fonctionne-t-il sur ordinateur ?

    Oui, il peut ouvrir WhatsApp Web ou l'application desktop selon le navigateur et l'appareil. Testez toujours avant une campagne.

    Puis-je suivre les clics ?

    Un lien wa.me brut ne donne pas de statistiques dans AfroTools. Pour mesurer une campagne, utilisez un lien court ou votre propre système d'analytics en respectant la vie privée.
