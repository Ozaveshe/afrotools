Questions fréquentes

Qu’est-ce qu’un lien WhatsApp ?

C’est un lien wa.me qui ouvre directement une conversation WhatsApp avec votre numéro, sans que le client ait besoin de l’enregistrer.

Puis-je ajouter un message par défaut ?

Oui, vous pouvez définir un message pré-rempli que le client verra automatiquement en ouvrant la conversation.

Le lien fonctionne-t-il sur mobile et desktop ?

Oui, le lien fonctionne sur tous les appareils : WhatsApp sur mobile ou WhatsApp Web sur ordinateur.
