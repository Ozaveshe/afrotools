Le calculateur estime les droits de douane, la TVA, les prelevements et le cout rendu total pour les pays couverts. Les montants restent indicatifs : la classification SH, les exemptions, les accords commerciaux et la valorisation retenue par la douane peuvent changer le resultat final.

Comment l'utiliser

  1. Choisissez le pays de destination.
  2. Selectionnez une categorie ou recherchez un code SH proche.
  3. Saisissez la valeur FOB, le transport et l'assurance.
  4. Lisez le detail : CIF, droit, prelevements, TVA, total USD et total local.

Avant une importation reelle, confirmez le code SH et le taux applicable avec votre transitaire ou l'autorite douaniere du pays.

FAQ Droits de douane

Questions courantes

Les montants sont-ils officiels ?

Non. L'outil fournit une estimation a partir des donnees disponibles. Le montant final appartient a l'autorite douaniere.

Pourquoi le transport change-t-il le total ?

De nombreuses administrations calculent les droits sur la valeur CIF, qui inclut marchandises, assurance et fret.

A quoi sert le code SH ?

Le code SH classe le produit et determine le taux douanier. Un code approximatif peut donner une estimation incorrecte.

Avertissement : Les taux sont indicatifs et doivent etre verifies avec l'autorite douaniere ou un transitaire avant toute importation.
