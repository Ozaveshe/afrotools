Signaler une erreur de calcul

Ce qui est calculé

Les gains bruts additionnent les lignes saisies. Le versement ajoute les remboursements. Le net retranche toutes les retenues. La contribution employeur s’ajoute au coût sans réduire le net.

Ce qui ne l’est pas

Aucun barème PAYE, taux de pension, plafond d’assurance, salaire minimum, congé ou droit légal n’est déduit automatiquement.
