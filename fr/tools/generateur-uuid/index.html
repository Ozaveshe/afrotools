Générateur UUID local

Créez, validez et normalisez des UUID v4 ou v7 sans quitter votre navigateur. La page montre le format, la version, les caractères modifiés et les choix pratiques pour bases de données, API et tests.

Copiez ce bloc dans une note technique, une revue de migration ou une fiche de test.

Diagnostic rapide

Lecture immédiate du choix d'identifiant.
Générez ou validez une valeur pour afficher le diagnostic.
  • Utiliser UUID v4 pour une valeur aléatoire simple.
  • Utiliser UUID v7 quand le tri chronologique aide les index.
  • Ne pas mettre de secret métier dans un UUID.

Méthodologie

Comment les résultats sont calculés.
  • UUID v4 est généré avec crypto.randomUUID() quand le navigateur le prend en charge, puis avec crypto.getRandomValues() en secours local.
  • UUID v7 assemble un horodatage en millisecondes, la version 7, la variante RFC et des octets aléatoires fournis par Web Crypto.
  • La validation supprime les enveloppes courantes, vérifie 32 caractères hexadécimaux, reconstruit le format canonique et compare les changements visibles.
  • Le résumé et les téléchargements sont produits dans le navigateur. Aucun fichier n'est envoyé à AfroTools.

Confidentialité et limites

Information générale, pas une garantie d'architecture.

Traitement local : les valeurs générées, collées, copiées ou téléchargées restent dans votre navigateur pendant cette session.

Avertissement : ce générateur aide à préparer des identifiants et à vérifier leur forme. Il ne remplace pas une revue de sécurité, une décision d'architecture ou les règles de votre base de données. Vérifiez vos contraintes d'unicité, de journalisation et de confidentialité dans l'environnement réel.
