Testeur de regex en ligne

Le testeur de regex d'AfroTools vous permet de creer et debuguer vos expressions regulieres en temps reel. Chaque correspondance est coloree et les groupes de capture sont identifies automatiquement. Ideal pour valider des formats de donnees, extraire des informations ou construire des filtres complexes.

L'outil supporte les drapeaux JavaScript (g, i, m, s, u) et fournit une reference rapide des metacaracteres. Partagez vos expressions via URL pour collaborer facilement avec votre equipe.

Questions frequentes

Quels moteurs regex sont supportes ?

L'outil utilise le moteur regex JavaScript natif du navigateur. Il supporte les drapeaux g, i, m, s, u et les groupes de capture nommes.

Comment partager une regex avec mon equipe ?

Copiez l'URL de l'outil qui inclut votre expression encodee. Partagez ce lien avec vos collegues pour qu'ils voient exactement la meme regex et les memes tests.

L'outil supporte-t-il les lookbehind ?

Oui, si votre navigateur le supporte. Les lookbehind sont disponibles dans Chrome, Edge et Firefox recents. Safari a un support limite des lookbehind.
