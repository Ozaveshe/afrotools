Testeur Regex

Groupes visibles

Attention

Contrôlez les cas limites avant d'appliquer une regex sur des données réelles.

Méthodologie

Detail du calcul: le test execute votre expression régulière contre le texte fourni, puis affiche correspondances, groupes et options utiles.

Limites

Un test positif ne garantit pas que la regex sera sure en production. Vérifiez performance, encodage, entrées vides et caractères inattendus.
