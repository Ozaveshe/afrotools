Méthode

  1. Le fichier est chargé localement avec FileReader, puis affiché en aperçu.
  2. Le navigateur dessine l'image dans un canvas, avec une largeur maximale facultative.
  3. canvas.toBlob produit le format cible quand il est pris en charge : PNG, JPEG ou WebP.
  4. Le poids affiché vient du Blob converti, avant téléchargement.

Vérification de page : 16 mai 2026. Références de benchmark : flux de conversion sur CloudConvert, conversion et optimisation locale inspirees de Squoosh, conversion image sur Canva et Adobe Express.

Limites

Le canvas du navigateur ne convertit pas tous les formats professionnels. Les fichiers HEIC, TIFF, RAW ou SVG complexes peuvent ne pas se charger. Le passage vers JPEG supprime la transparence, donc un fond est applique. PNG conserve mieux les aplats et la transparence, tandis que JPEG et WebP peuvent produire des fichiers plus legers selon l'image.

Confidentialité et lots

Les images restent sur votre appareil pendant la session. La file accepte plusieurs images pour préparer un lot, mais l'interface convertit l'image sélectionnée afin que vous puissiez vérifier le rendu avant téléchargement.
