Méthode

  1. L'image est chargee localement avec FileReader ou un objet URL, puis dessinee dans le canvas.
  2. Les contrôles construisent une chaine de filtre canvas : luminosite, contraste, saturation, noir et blanc et sepia.
  3. Le mode avant/après affiche l'original d'un côté et le rendu filtré de l'autre pour contrôler l'effet avant export.
  4. Le téléchargement crée un PNG propre depuis le canvas, sans le trait de comparaison.

Vérification de page : 16 mai 2026. Références de benchmark : effets photo sur Canva, filtres photo sur Adobe Express, edition locale inspiree de Squoosh, et filtres canvas documentes par MDN.

Limites

Les filtres canvas sont rapides et pratiques, mais ils ne remplacent pas une suite photo avancee pour la retouche fine, les masques, les profils couleur ou les corrections de peau. Le rendu final peut varier legerement selon le navigateur et l'écran.

Confidentialité

Cette page ne fait pas d'envoi serveur pour traiter votre image. Pour les photos sensibles, gardez tout de même une copie originale séparée et vérifiez le fichier exporté avant de le partager.
