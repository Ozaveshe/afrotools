Décodeur JWT en ligne

Le décodeur JWT d'AfroTools vous permet d'inspecter rapidement un token JSON Web Token. Visualisez l'en-tête avec l'algorithme utilisé, le payload avec les revendications et les dates d'expiration. Cet outil aide au développement d'applications avec authentification OAuth ou OpenID Connect.

L'outil fonctionne entièrement hors ligne une fois chargé. Aucun token n'est envoyé à un serveur. Collez votre JWT et obtenez instantanément une vue claire et structurée de son contenu.

Questions fréquentes

Qu'est-ce qu'un token JWT ?

Un JWT (JSON Web Token) est un standard ouvert pour transmettre des informations entre deux parties sous forme de jeton signé numériquement.

Le décodeur vérifie-t-il la signature JWT ?

L'outil décode et affiche le contenu du JWT mais ne vérifie pas la signature cryptographique. Pour la vérification de signature, utilisez une bibliothèque côté serveur.

Quels algorithmes JWT sont supportés ?

L'outil décode les algorithmes JWT courants : HS256, HS384, HS512, RS256, RS384, RS512, ES256, ES384, ES512, PS256, PS384, PS512 et EdDSA.
