Decodeur JWT en ligne

Le decodeur JWT d'AfroTools vous permet d'inspecter rapidement n'importe quel token JSON Web Token. Visualisez l'en-tete avec l'algorithme utilise, le payload avec toutes les revendications, et verifiez les dates d'expiration. Outil essentiel pour le developpement d'applications avec authentification OAuth ou OpenID Connect.

L'outil fonctionne entierement hors ligne une fois charge. Aucun token n'est envoye a un serveur. Collez votre JWT et obtenez instantanement une vue claire et structuree de son contenu.

Questions frequentes

Qu'est-ce qu'un token JWT ?

Un JWT (JSON Web Token) est un standard ouvert pour transmettre des informations de maniere securisee entre deux parties sous forme de jeton signe numeriquement.

Le decodeur verifie-t-il la signature JWT ?

L'outil decode et affiche le contenu du JWT mais ne verifie pas la signature cryptographique. Pour la verification de signature, utilisez une bibliotheque cote serveur.

Quels algorithmes JWT sont supportes ?

L'outil decode tous les algorithmes JWT courants : HS256, HS384, HS512, RS256, RS384, RS512, ES256, ES384, ES512, PS256, PS384, PS512 et EdDSA.
