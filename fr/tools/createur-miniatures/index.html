Méthode

  1. Choisissez le format selon la plateforme : 16:9 pour YouTube classique, vertical pour Shorts et story, carré ou vertical pour réseaux sociaux.
  2. Placez le titre principal dans la zone centrale. La ligne pointillée montre une marge de sécurité pour les recadrages et les commandes d'interface.
  3. Ajoutez un portrait, un produit ou une image de fond si cela aide le sujet. Sans image, le canvas garde un repere visuel propre.
  4. Le bouton de téléchargement crée un PNG depuis le canvas. La zone de sortie garde le titre et la légende prêts à copier.

Conseils de design

  • Gardez un titre court, très contraste, avec une seule promesse ou question.
  • Utilisez le badge pour une preuve rapide : prix, episode, avant/après, direct, nouveau ou urgent.
  • Evitez les textes colles aux bords. Les plateformes peuvent afficher des boutons, durées ou menus par-dessus.
  • Testez l'aperçu à petite taille avant de publier : si le titre n'est pas lisible, simplifiez.

Confidentialité et limites

L'image importée est lue localement avec FileReader puis dessinée dans le canvas. Aucune image n'est envoyée à AfroTools depuis cette page. Ce créateur sert de point de départ rapide : vérifiez toujours les droits d'image, la lisibilité, les règles de marque et les exigences exactes de chaque plateforme avant publication.
