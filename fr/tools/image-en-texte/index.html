Méthode et fraîcheur

  1. Le fichier est lu par le navigateur avec FileReader, puis affiche en aperçu local.
  2. Pour une image, le moteur OCR local Tesseract.js est appele si la bibliotheque est disponible dans la page.
  3. Pour un PDF, PDF.js rend la première page sur canvas, puis l'OCR lit cette image rendue.
  4. Le texte obtenu reste éditable, puis peut être copie ou telecharge au format TXT.

Vérification de page : 16 mai 2026. Sources techniques : Tesseract.js pour l'OCR dans le navigateur et PDF.js pour le rendu PDF cote navigateur.

Limites et avertissement

Ce service est une aide pratique, pas une certification de lecture. Les documents manuscrits, compresses, inclines, très sombres ou multi-colonnes peuvent produire des erreurs. Les PDF scannes page par page peuvent demander une vérification manuelle. Si le moteur OCR ne se charge pas dans votre navigateur, aucun faux résultat n'est genere : utilisez la zone de texte comme filet de secours.

Pour un document officiel, financier, médical ou juridique, comparez toujours le texte extrait avec l'original avant de l'utiliser.

Confidentialité et traitement local

Le traitement se fait dans votre navigateur. AfroTools ne recoit pas vos fichiers depuis cette page. Les bibliotheques chargees servent à previsualiser et analyser le fichier localement, puis le TXT est cree sur votre appareil.
