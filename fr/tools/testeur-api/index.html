Testeur d'API REST en ligne

Le testeur d'API d'AfroTools est une alternative legere a Postman directement dans votre navigateur. Envoyez des requetes HTTP avec des en-tetes personnalises, un corps JSON et differents modes d'authentification. Visualisez les reponses formatees avec les codes de statut, les en-tetes et les temps de reponse en un coup d'oeil.

L'historique des requetes est sauvegarde localement pour retrouver rapidement vos tests precedents. Aucune donnee ne transite par nos serveurs, vos cles API et tokens restent confidentiels sur votre machine.

Questions frequentes

Comment tester une API protegee ?

Ajoutez vos tokens Bearer, cles API ou identifiants Basic Auth dans la section authentification. Toutes les donnees restent locales sur votre navigateur.

Mes cles API sont-elles en securite ?

Oui, toutes les requetes sont envoyees directement depuis votre navigateur. Aucune donnee ne transite par les serveurs d'AfroTools.

Quels types de requetes sont supportes ?

GET, POST, PUT, PATCH, DELETE, HEAD et OPTIONS avec en-tetes personnalises et differents types de corps de requete (JSON, form-data, texte brut).
