Explorer le tableau periodique des elements

Notre tableau periodique interactif va au-dela d un simple outil de reference. Explorez les 118 elements avec leurs proprietes detaillees : masse atomique, configuration electronique, points de fusion et d ebullition, electronegativite et rayon atomique.

Ce qui rend notre outil unique est le contexte africain : decouvrez quels elements sont extraits en Afrique, les principaux pays producteurs et le role du continent dans l approvisionnement mondial en mineraux. Un mode quiz teste vos connaissances.

Questions frequemment posees

Quelles proprietes sont affichees ?

Masse atomique, numero atomique, configuration electronique, electronegativite, points de fusion et d ebullition, rayon atomique, energie d ionisation, categorie et etat a temperature ambiante.

Quel est le lien avec l Afrique ?

L Afrique est un producteur majeur de nombreux elements : or (Ghana, Afrique du Sud), cobalt (RDC), platine (Afrique du Sud), diamant (Botswana), uranium (Niger). L outil met en lumiere ces connexions.

Y a-t-il un mode quiz ?

Oui. Le mode quiz teste vos connaissances : identifiez les elements par symbole, numero atomique ou proprietes. Ideal pour reviser avant un examen de chimie.
