Convertir HTML en PDF

L’outil de conversion HTML en PDF d’AfroTools transforme du code HTML collé ou du contenu enrichi en document PDF.

Le rendu dépend du HTML, des images, des polices et des styles disponibles dans le navigateur. Relisez les marges, sauts de page et caractères spéciaux avant d’utiliser le fichier.

Questions fréquentes

Puis-je convertir directement l’URL d’un site ?

Non. Cette page fonctionne avec du code HTML collé ou du contenu enrichi saisi dans l’éditeur; la capture d’une page web en direct nécessite un rendu côté serveur.

Les images et styles sont-ils conservés ?

Ils peuvent être repris si le navigateur y a accès, mais le rendu final doit toujours être relu avant partage.

Puis-je personnaliser le format de sortie ?

Oui, ajustez la taille (A4, Letter), les marges et l’orientation (portrait ou paysage).
