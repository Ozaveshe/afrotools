Méthodologie

La page lit l'image avec FileReader, la dessine dans un canvas aux dimensions choisies, puis exporte le rendu via le navigateur. Quand le ratio est verrouille, la hauteur ou la largeur est recalculee depuis le ratio original.

  • PNG: utile pour logos, transparence et captures propres.
  • JPEG: utile pour photos et petits fichiers.
  • WebP: utile pour sites web modernes quand le navigateur le supporte.

Confidentialité et avertissement

L'image source reste local-only dans votre navigateur. L'export canvas retire généralement les metadonnees EXIF/GPS. Vérifiez les limites exactes du portail avant soumission.

Pourquoi l'image peut-elle devenir floue ?

Si vous agrandissez une petite image, le navigateur doit inventer des pixels. Pour garder la nettete, partez d'une source plus grande que la sortie.

Quel format choisir pour une photo ?

JPEG ou WebP sont souvent plus legers pour une photo. PNG est meilleur pour logo, capture, texte ou transparence.

Est-ce que cette page traite les lots ?

Cette page française traite un fichier a la fois. Le batch workflow explique comment garder les reglages, et l'outil anglais complet propose un flux plus avance.
