Calculateur de surestaries

Estimez le risque de frais portuaires apres les jours de franchise et preparez une marge de securite avant enlevement, inspection ou dedouanement.

Utilisation prudente

Les surestaries peuvent transformer une marge import en perte lorsque les documents, le paiement, l inspection ou le transport interieur prennent du retard. Cette page sert a cadrer le risque, pas a remplacer la facture du port ou de la compagnie maritime.

Verification recommandee. Confirmez les jours gratuits, le type de conteneur, les frais de detention et les conditions locales avec l agent maritime, le terminal ou votre transitaire.
