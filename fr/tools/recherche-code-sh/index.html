À propos

Trouvez le code SH (Système Harmonisé) de vos marchandises pour le dédouanement en Afrique. Recherchez par nom de produit ou par code.

Le code SH est essentiel pour déterminer les droits de douane et les réglementations applicables à vos marchandises. Cet outil vous aide à trouver le bon code pour vos opérations commerciales en Afrique.

Questions fréquentes

Comment trouver mon code SH ?

Recherchez par nom de produit ou naviguez dans les chapitres du Système Harmonisé. L’outil affiche le code à 6 chiffres et les sous-positions nationales.

Pourquoi le code SH est-il important ?

Il détermine les droits de douane, les restrictions et les documents requis pour l’importation et l’exportation de vos marchandises.

Les codes sont-ils les mêmes partout ?

Les 6 premiers chiffres sont universels. Les pays africains ajoutent des chiffres supplémentaires pour leurs classifications nationales.
