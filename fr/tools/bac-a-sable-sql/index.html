Bac a sable SQL en ligne

Le bac a sable SQL d'AfroTools vous offre un environnement SQLite complet directement dans le navigateur. Executez des requetes CREATE, INSERT, SELECT, UPDATE et DELETE sans aucune installation. Ideal pour apprendre le SQL, tester des requetes ou prototyper des schemas de base de données.

La base de données est ephemere et stockee en memoire. Vos données restent confidentielles et disparaissent a la fermeture de l'onglet. Importez des fichiers CSV pour travailler avec vos propres données.

Questions fréquentes

Les données sont-elles sauvegardees ?

Non, la base SQLite est ephemere et stockee en memoire. Toutes les données disparaissent quand vous fermez l'onglet. Exportez vos résultats avant de quitter.

Puis-je importer des données CSV ?

Oui, vous pouvez importer des fichiers CSV pour créer des tables et travailler avec vos propres données dans l'environnement sandbox.

Puis-je créer des tables et des index ?

Oui, l'environnement sandbox supporte toutes les commandes DDL de SQLite : CREATE TABLE, CREATE INDEX, ALTER TABLE, DROP TABLE, et plus encore.
