Constructeur d'expressions Cron

Le constructeur cron d'AfroTools vous aide a creer des expressions cron visuellement sans memoriser la syntaxe. Selectionnez la frequence, les jours et les heures, et l'expression est generee automatiquement. La previsualisation affiche les prochaines dates d'execution pour verifier que votre planification est correcte.

L'outil supporte la syntaxe cron standard a 5 champs et la syntaxe etendue a 6 champs avec les secondes. Copiez l'expression generee directement dans vos fichiers crontab, GitHub Actions ou pipelines CI/CD.

Questions frequentes

Qu'est-ce qu'une expression cron ?

Une expression cron est une chaine de 5 champs (minute, heure, jour du mois, mois, jour de la semaine) qui definit un calendrier de taches planifiees sur les systemes Unix.

Puis-je utiliser l'expression dans GitHub Actions ?

Oui, la syntaxe cron generee est compatible avec GitHub Actions, crontab Unix, GitLab CI, Jenkins et la plupart des systemes de planification de taches.

Quelle est la difference entre cron 5 et 6 champs ?

Le format 5 champs (minute, heure, jour, mois, jour de semaine) est le standard Unix. Le format 6 champs ajoute les secondes en premiere position, utilise par Spring et Quartz.
