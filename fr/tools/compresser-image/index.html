Methode

  1. Le fichier est charge localement avec FileReader, puis dessine dans un canvas.
  2. La largeur maximale redimensionne l'image si elle depasse le seuil choisi.
  3. Le format JPEG ou WebP utilise le niveau de qualite du navigateur. PNG reste surtout utile quand vous voulez garder la transparence.
  4. Le poids final est estime depuis le Blob produit par canvas.toBlob avant telechargement.

Verification de page : 16 mai 2026. References de benchmark : compression WebP, PNG et JPEG chez TinyPNG, compression locale dans le navigateur avec Squoosh, redimensionnement et export sur Canva et Adobe Express.

Limites

La compression canvas ne remplace pas les encodeurs avances de services specialises. Le navigateur peut retirer certaines metadonnees, modifier les profils couleur ou ne pas prendre en charge WebP dans tous les contextes. Pour les logos transparents, testez PNG et WebP avant de choisir le fichier final.

Confidentialite et file de lots

Les images restent sur votre appareil pendant la session. La file accepte plusieurs images pour preparer un lot, mais l'interface compresse l'image selectionnee afin que vous puissiez verifier chaque resultat avant telechargement.
