Editeur Markdown en ligne

L'editeur Markdown d'AfroTools offre un environnement d'edition avec apercu en temps reel. Supportant la syntaxe GitHub Flavored Markdown, il gere les tableaux, le code en surbrillance, les listes de taches et plus encore. Exportez votre document en HTML propre ou copiez le rendu directement.

L'outil est ideal pour rediger des README GitHub, de la documentation technique ou des articles de blog. Tout le traitement est local et aucun contenu n'est envoye a un serveur.

Questions frequentes

Quelle syntaxe Markdown est supportee ?

L'editeur supporte GitHub Flavored Markdown (GFM) incluant les tableaux, le code en surbrillance, les listes de taches, les notes de bas de page et les emojis.

Puis-je exporter en HTML ?

Oui, vous pouvez exporter votre document Markdown en HTML propre et semantique. Copiez le code HTML ou telechargez-le directement.

L'editeur supporte-t-il les tableaux ?

Oui, l'editeur supporte les tableaux Markdown avec alignement des colonnes (gauche, centre, droite). La coloration syntaxique fonctionne pour plus de 20 langages de programmation.
