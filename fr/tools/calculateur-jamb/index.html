Calculer votre score JAMB Aggregate

Le score JAMB Aggregate est essentiel pour les admissions dans les universites nigerianes. Chaque universite utilise sa propre formule combinant les resultats JAMB UTME et les notes de Post-UTME ou O-Level. Notre calculateur prend en charge les formules de UNILAG, UI, OAU, UNN et autres.

Entrez vos scores JAMB UTME par matiere, selectionnez l universite visee et obtenez instantanement votre score aggregate. Comparez votre resultat aux seuils d admission des annees precedentes pour evaluer vos chances.

Questions frequemment posees

Comment est calcule le score JAMB Aggregate ?

Le calcul varie selon l universite. Il combine les scores JAMB UTME (sur 400) et les resultats Post-UTME ou O-Level selon une formule ponderee propre a chaque etablissement.

Quel score JAMB faut-il pour entrer a l universite ?

Le seuil varie : UNILAG exige generalement 200+, UI 180+. Les programmes competitifs comme medecine exigent des scores plus eleves.

Les scores JAMB sont-ils valables combien de temps ?

Les resultats JAMB UTME sont valables pour un seul cycle d admission. Vous devez repasser l examen chaque annee si vous n obtenez pas d admission.
