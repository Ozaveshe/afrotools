Méthodologie

Le générateur dessine une image source ou des initiales sur canvas, puis recree les tailles utiles pour navigateur et mobile. Les PNG sont exportes localement pour que vous puissiez les placer dans le dossier racine du site.

  • 16x16 et 32x32 pour les onglets et favoris.
  • 48x48 et 64x64 pour les usages haute densite.
  • 180x180 pour un raccourci mobile simple.

Batch workflow

Pour plusieurs sites, créez une queue par marque: nom du site, source image, couleur de fond, droits verifes et date. Exportez les tailles PNG puis stockez le snippet avec le projet afin de ne pas melanger les favicons.

Confidentialité et avertissement

Votre image reste locale pendant la génération. Le bouton sauver garde seulement un brouillon dans ce navigateur. Confirmez toujours que vous avez le droit d'utiliser le logo, surtout pour un client ou une marque publique.

Un favicon doit-il être carré ?

Oui, un format carré est le plus fiable. Si votre image est rectangulaire, gardez une marge suffisante autour du symbole principal.

Dois-je utiliser ICO ou PNG ?

PNG couvre les navigateurs modernes. Si vous avez besoin d'un fichier ICO traditionnel, ouvrez l'outil anglais complet ou utilisez votre chaine de build habituelle.

Comment savoir si l'icone est bonne ?

Regardez la version 16 pixels. Si le symbole devient flou ou illisible, simplifiez les couleurs, augmentez la marge ou utilisez seulement deux initiales.
