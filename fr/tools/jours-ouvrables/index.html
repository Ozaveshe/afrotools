Methodologie

        
Methodology: l'outil parcourt les dates du scenario, applique les jours de week-end choisis, compare les dates aux jours feries de depart et aux dates personnalisees, puis calcule le nombre de jours ouvrables restants.

        
    
          
  • Regles appliquees: dates inclusives par defaut pour le mode entre deux dates; le mode echeance ajoute uniquement les jours qui ne sont pas exclus.
    • 
          
  • Avertissement: ce resultat est une aide de planification non officielle. Il ne remplace pas une autorite publique, un contrat, un tribunal, un service RH ou un conseil professionnel.
    • 
          
  • Fraicheur: revu en 2026; les jours feries mobiles et annonces speciales doivent etre verifies avant action.
    • 
          
  • Confidentialite: le calcul se fait dans le navigateur; la sauvegarde reste locale.

Questions frequentes

        
Le samedi peut-il etre ouvrable ?
Oui. Decochez samedi dans les jours exclus si votre equipe, commerce ou contrat utilise une semaine de six jours.

        
Pourquoi ajouter des dates personnalisees ?
Certains jours feries sont annonces tard, varient par region ou dependent d'une entreprise. Ajoutez-les pour garder le calcul proche de votre realite.

        
Le resultat est-il officiel ?
Non. Pour la paie, un depot fiscal, un tribunal ou une obligation contractuelle, verifiez la date finale avec l'autorite officielle ou le document applicable.
