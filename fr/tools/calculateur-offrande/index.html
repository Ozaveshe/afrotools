Questions frequentes

Dois-je calculer sur le brut ou le net ?

C'est une decision personnelle ou communautaire. Beaucoup de personnes utilisent le net recu pour proteger le budget familial; d'autres utilisent le brut par conviction. L'outil montre la base choisie pour que le resume reste clair.

Une offrande est-elle la meme chose qu'une dime ?

Dans beaucoup d'eglises, la dime designe un pourcentage regulier, souvent 10%, tandis que l'offrande est un don libre. Les pratiques varient selon l'eglise, la confession et le pays.

Ce calculateur peut-il remplacer un conseil pastoral ou fiscal ?

Non. Il aide a poser les chiffres. Pour une question doctrinale, demandez conseil a votre responsable religieux. Pour les impots, utilisez la source officielle de votre administration fiscale.
