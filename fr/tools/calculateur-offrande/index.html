Questions fréquentes

Dois-je calculer sur le brut ou le net ?

C'est une décision personnelle ou communautaire. Beaucoup de personnes utilisent le net reçu pour proteger le budget familial; d'autres utilisent le brut par conviction. L'outil montre la base choisie pour que le résumé reste clair.

Une offrande est-elle la même chose qu'une dîme ?

Dans beaucoup d'églises, la dîme designe un pourcentage régulier, souvent 10%, tandis que l'offrande est un don libre. Les pratiques varient selon l'église, la confession et le pays.

Ce calculateur peut-il remplacer un conseil pastoral ou fiscal ?

Non. Il aide à poser les chiffres. Pour une question doctrinale, demandez conseil a votre responsable religieux. Pour les impôts, utilisez la source officielle de votre administration fiscale.
