Méthodologie et limites

  • Base brute = revenu brut declare.
  • Base nette = revenu brut - déductions essentielles.
  • Base libre = montant que vous saisissez comme disponible après budget.
  • Dîme = base x pourcentage. Total = dîme + offrande fixe.

Checklist de prudence

  • Ne sacrifiez pas nourriture, loyer, transport ou soins essentiels pour suivre un chiffre.
  • Clarifiez si votre communauté parle de brut, net, premier fruit ou don libre.
  • Gardez un reçu ou une note personnelle si vous suivez les dons annuels.
  • Revoyez le montant après changement de salaire, dette, maladie ou responsabilité familiale.

Questions fréquentes

La base brute ou nette est-elle la bonne ?

Les pratiques varient selon les églises, familles et convictions. L'outil affiche les deux logiques pour faciliter une décision informee, pas pour imposer une réponse.

Puis-je utiliser un autre pourcentage ?

Oui. Modifiez le taux selon votre tradition ou votre engagement personnel. Le taux de 10% est seulement le point de départ par défaut.

Mes données sont-elles envoyees ?

Non. Les calculs se font dans le navigateur. La sauvegarde locale reste sur cet appareil tant que le stockage du navigateur n'est pas efface.
