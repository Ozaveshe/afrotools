# V3 Targeted Copy Fixes

This was a targeted cleanup, not a full copy audit rerun.

## Files Improved

- `salary-tax/index.html`
- `tools/mobile-money-fees/index.html`

## Repeated Language Problems Removed

- Overconfident speed claims.
- Hype around "real" tables and instant precision.
- Generic tool-suite phrasing.
- Long disclaimers that made the tool feel less practical.
