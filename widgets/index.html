How it works.

The widget experience is meant to be simple enough for a CMS page and flexible enough for a custom site.

1

Choose

Search the library by topic, country, or workflow. Open the gallery when you want to compare options side by side.

2

Preview

Test the widget in light or dark mode and check the iframe fallback before placing it on your page.

3

Embed

Copy the script snippet for the styled widget or copy the iframe code for publishing systems with stricter rules.
