How it works.

The widget experience is meant to be simple enough for a CMS page and flexible enough for a custom site, with script and iframe paths that keep partner publishing friction low.

1

Choose

Search the library by topic, country, or workflow. Open the gallery when you want to compare options side by side.

2

Preview

Test the widget in light or dark mode and check the iframe fallback before placing it on your page.

3

Embed

Copy the script snippet for the styled widget or copy the iframe code for publishing systems with stricter rules. Free widgets keep visible AfroTools attribution.

Widget questions.

Pick the free embed when attribution is fine. Ask for Widget Pro or a custom build when the widget needs partner styling, analytics, lead capture, or a source-reviewed workflow.

Can I embed AfroTools widgets for free?

Yes. Public widgets can be embedded with visible AfroTools attribution. For white-label styling, setup support, analytics, or lead capture, request a Widget Pro pilot.

What kind of site should use the widget library?

Widgets fit publisher articles, school sites, HR resources, fintech education pages, association portals, business blogs, and country guides where visitors need a quick calculator.

Do widgets need an API key?

Most public embeds do not need an API key. API-backed, high-volume, or partner data integrations should use the AfroTools API or a custom calculator pilot.
