FAQ

Is it free? ▼
Yes, the widget library is free to use. No sign-up, no API key, and no usage caps for the public widgets. Free widgets include a small Powered by AfroTools footer.
Will it slow my website? ▼
No. The loader is about 9KB, loads asynchronously, and lazy-loads widgets with IntersectionObserver so they only initialize when visible. Zero render-blocking.
Can I customize the colors? ▼
Yes! Use data-afrotools-theme="light" or "dark", data-afrotools-accent="#yourcolor", and data-afrotools-width to control the max width.
Do you track my visitors? ▼
We only track anonymous widget load counts (which widget loaded and the referring domain). No personal data, no cookies, no fingerprinting.
Can I use multiple widgets on one page? ▼
Absolutely. Add as many data-afrotools divs as you want. The single embed.js script handles all of them. Each widget uses Shadow DOM for complete CSS isolation.
What about the "Powered by AfroTools" link? ▼
Free widgets include a small footer link on each widget. It helps us keep the library free, and it gives publishers a transparent source link back to the full tool on AfroTools.com.
Do the calculators produce accurate results? ▼
Yes. Every widget uses the exact same calculation logic as the full tool on AfroTools.com. PAYE calculators use official 2025/2026 tax bands from each country's revenue authority.
