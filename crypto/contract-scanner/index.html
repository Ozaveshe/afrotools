How to interpret the result

Syntax only

All listed chains use the same basic EVM text shape. This check deliberately does not claim checksum validation.

Exact reviewed records only

A record appears only when both the selected chain and normalized address match a reviewed registry entry with a cited HTTPS source.

Selected explorer only

For valid syntax, the result links only to the selected chain’s explorer. Opening an explorer discloses the address to that external service.

Questions

Is “no reviewed record” a clearance?

No. Registry coverage is intentionally small and incomplete. Absence is not evidence of safety.

What should I review manually?

On the selected explorer, verify the exact chain and address yourself. Contract source, permissions, holders, liquidity and transaction behavior are outside this tool.

Is the address sent anywhere?

Not by AfroTools. The local registry is loaded without the address. The address leaves the page only if you deliberately open the selected external explorer link or download your local text record.
