Questions about the evidence organizer

Does this tool check a scam database or validate a wallet?

No. It only organizes information you enter and never labels a person, wallet, platform or transaction safe, fraudulent or verified.

Does AfroTools receive the incident details?

No. The organizer and exports run in your browser without a lookup, submission or storage request.

What should never be entered?

Never enter a private key, seed phrase, password, one-time code or other secret that can control an account or wallet.
