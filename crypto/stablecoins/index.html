What the snapshot measures

AfroTools requests three fixed CoinGecko asset IDs through a dedicated server function. Each asset must include positive USD and local reference prices plus a provider timestamp no more than 30 minutes old. A row that fails those checks is withheld. If no row survives, the table closes safely.

USD peg distance is derived only as the percentage difference between the CoinGecko USD reference and 1 USD. It can help describe the provider snapshot, but it is not an exchange premium, platform spread, fee, savings yield or forecast.

Use the right evidence

  • These values are indicative provider references, not executable quotes.
  • Verify the exact platform, payment method, network, fees and withdrawal conditions before acting.
  • NGN and ZAR are shown because they are the African quote currencies verified directly with this provider contract.
  • Exports include the source, scope, quote currency and timestamps. PDF is not offered because structured CSV and JSON preserve the receipt.
  • This is market information, not financial advice.

Stablecoin reference FAQ

Does the lowest peg distance identify the best stablecoin?

No. A single provider price does not assess issuer reserves, redemption access, smart-contract risk, network availability, regulation, liquidity or your platform’s executable price.

Why are there no platform rankings?

This app has no current, complete platform-quote source for USDT, USDC and DAI. It therefore does not invent platform rates, trust scores, fees, processing times, spreads or recommendations.

What happens when the provider is unavailable?

No expired cache, hard-coded fallback or estimated local conversion is shown. The page reports that fresh provider data is unavailable and keeps exports disabled until a fresh snapshot loads.
