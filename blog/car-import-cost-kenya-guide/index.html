Kenya Car Import Cost Guide

The Kenya Car Import Cost Calculator is built for the Japan to Mombasa workflow but also supports UAE, UK, South Africa, and local dealer comparisons.

It works best as an early budgeting tool, not a substitute for the current KRA rule pack. Customs value, CRSP changes, the eight-year rule, and KEBS compliance checks can all change what finally lands on the release file.

What users miss

  • KRA valuation can differ from the seller invoice.
  • Excise, VAT, IDF, and RDL stack on top of duty.
  • KEBS inspection and document checks are not optional.
  • Nairobi, Kisumu, and Eldoret delivery add meaningful cost.
  • Age and steering failures can block the import regardless of price.

Official sources to check before paying

  • KRA: Procedures for Motor Vehicle Importation
  • KRA: current CRSP notice for used motor vehicles
  • KRA: depreciation treatment for used motor vehicles

Use the calculator as a planning estimate and verify the latest KRA and KEBS position before you commit funds.
