Developer Tools FAQ

Are these tools free?
Yes. The core developer utilities are free and do not require an account.
Is my data sent to AfroTools?
Most utilities process data in your browser. API Tester sends the request directly from your browser to the URL you choose, so production secrets should still be handled carefully.
Why are African APIs and USSD highlighted?
Those are the workflows where AfroTools can be more useful than generic global utilities: mobile money, telecom, identity, USSD menus, local payment constraints and deployment choices.
