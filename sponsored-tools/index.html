Trust rules

AfroTools sponsorships are built to create revenue without making the tools less useful or less honest for users.

Calculation logic stays independent

A sponsor can fund placement, but cannot change formulas, sources, or outputs to favor its product.

Paid placement is labeled

Sponsored blocks are visibly marked so the page remains useful and honest for the user.

Regulated claims need approval

Insurance, tax, payroll, credit, immigration, and education claims need review before launch.

Leads need explicit opt-in

Sponsor lead routing is available only when the user knowingly submits a form or request.
