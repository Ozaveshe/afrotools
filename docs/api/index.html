Available APIs

Three endpoints covering the core financial data infrastructure for Africa.

AfroFX API

Real-time exchange rates for 42 African currencies against major pairs (USD, EUR, GBP). 15-minute updates, historical data back to 2020, and crypto rates.

AfroFuel API

Fuel prices (petrol, diesel, LPG) for all 54 African countries. Weekly updates sourced from national petroleum regulators. Historical trends included.

Central bank policy rates, inflation figures, T-bill yields, and MPC meeting calendar for all 54 African countries. Updated within 24 hours of announcements.

Endpoint Reference

All endpoints return JSON. Authenticated via Bearer token in the Authorization header.

GET /api/forex?from=USD&to=NGN

Returns the latest exchange rate between two currencies. Supports all 42 African currencies plus USD, EUR, GBP, and major crypto.

Response { "from": "USD", "to": "NGN", "rate": 1580.25, "bid": 1578.50, "ask": 1582.00, "timestamp": "2026-03-15T10:30:00Z", "source": "CBN" }
GET /api/fuel?country=NG

Returns current fuel prices for a given country. Includes petrol (PMS), diesel (AGO), kerosene (DPK), and LPG where available.

Response { "country": "NG", "currency": "NGN", "petrol": { "price": 897.00, "unit": "litre" }, "diesel": { "price": 1420.00, "unit": "litre" }, "lpg": { "price": 1150.00, "unit": "kg" }, "updated": "2026-03-10" }
GET /api/rates?country=NG

Returns central bank policy rate, inflation, T-bill yields, and next MPC meeting date for a given country.

Response { "country": "NG", "central_bank": "CBN", "policy_rate": 27.50, "inflation": 31.7, "tbill_91d": 18.5, "tbill_182d": 20.1, "tbill_364d": 22.3, "next_mpc": "2026-05-20", "updated": "2026-03-01" }
