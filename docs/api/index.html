Quickstart

Three steps to your first API call.

1

Get an API Key

Sign up, verify your email, and generate your API key from the dashboard. Test keys return deterministic sandbox data with separate sandbox limits.

2

Make a Request

Pass your key via the x-api-key header or query parameter.

3

Get JSON Back

Every endpoint returns consistent, well-documented JSON responses.

# Calculate Nigerian tax on a 500,000 NGN salary curl -X POST https://afrotools.com/api/v1/tax/paye \ -H "x-api-key: afro_test_abc123" \ -H "Content-Type: application/json" \ -d '{"country":"NG","grossMonthly":500000}'

Status, Versioning, and Security

Current base URL: https://afrotools.com/api/v1. Service status is available at /api/status.

Current version

API v1 is the stable public version. Breaking changes will ship under a new URL version with at least 90 days of deprecation notice.

Sandbox keys

Test keys return deterministic sandbox data and have separate sandbox limits. They do not access live production data.

Protect production keys

Do not expose production keys in browser or mobile client code. Put calls behind your own backend and rotate keys from the dashboard when needed.

Six Products, One API

Everything you need to build African finance, data, and workflow routing products.

AfroTax API
Income tax, corporate tax, and withholding tax calculations for 54 African countries. Gross-to-net and net-to-gross modes.

AfroFX API
Real-time exchange rates for 42 African currencies against major pairs. 15-minute updates with historical data.
⛽
AfroFuel API
Fuel prices (petrol, diesel, LPG) for 54 African countries. Weekly updates from national petroleum regulators.
🏦
AfroRates API
Central bank policy rates, inflation, T-bill yields, and MPC meeting calendars for all 54 African countries.
💳
AfroVAT API
VAT/GST calculation engine for African countries. Add-on or inclusive modes with exemption support.
🤖
AfroTools AI Route API
Partner-safe task routing from user prompts to validated AfroTools tool and workflow links.
