Calculate Tax

Calculate PAYE salary tax for a supported country using gross-to-net or net-to-gross inputs.

POST https://afrotools.com/api/v1/tax/paye
Authentication required. Pass your API key via x-api-key header. Learn more

Request Body

Send a JSON body with the following parameters:

ParameterTypeDescription
country required string ISO 3166-1 alpha-2 country code (e.g., NG, KE, ZA).
grossMonthly * number Monthly gross salary in the country's local currency.
grossAnnual * number Annual gross salary. Use this instead of grossMonthly.
netMonthly * number Monthly net salary for net-to-gross calculation.
netAnnual * number Annual net salary for net-to-gross calculation.
pension optional boolean Include pension/social security deductions. Default: true.
regime optional string Country-specific tax regime where supported.

* Provide exactly one of grossMonthly, grossAnnual, netMonthly, or netAnnual.

Response Fields

FieldTypeDescription
countrystringISO country code.
currencystringISO currency code.
deductionsobjectPension, social security, and other deductions where applicable.
taxobjectTaxable income, net tax, and tax-band details.
resultobjectNet annual, net monthly, and effective rate output.
_metaobjectAPI version, response time, timestamp, and sandbox flag.

Error Responses

StatusCodeDescription
400invalid_countryThe country code is not supported.
400INVALID_SALARY_INPUTProvide exactly one salary input.
400MISSING_REQUIRED_FIELDA required field is missing.
401missing_api_keyNo API key was provided.
429rate_limit_exceededRate limit exceeded for your plan.
