API Changelog

Track new features, improvements, and fixes across all AfroTools API products.

March 15, 2026

New AfroVAT API Launch

  • New POST /api/vat endpoint for VAT/GST calculations
  • Supports exclusive (add) and inclusive (extract) calculation modes
  • Category-based rates: standard, food_basic, medical, education, digital, export
  • Multi-item invoice support for mixed-category calculations
  • Coverage for all 54 African countries
March 10, 2026

New AfroTax API Launch

  • New POST /api/tax endpoint for income, corporate, and withholding tax calculations
  • Gross-to-net and net-to-gross calculation modes
  • Pension/social security deductions included
  • GET /api/tax for listing countries and detailed country info
  • Full tax bracket data for all 54 African countries

Improve API Documentation Redesign
