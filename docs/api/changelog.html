Changelog

Track new features, improvements, and changes across all AfroTools API products.

Versioning policy. The AfroTools API uses URL-based versioning. The current version is v1. Breaking changes will always be released under a new version number with at least 90 days of deprecation notice.

v1.1 May 2026

Account reliability release for dashboard sync, AfroPoints, alerts, and the shared AI helper. No breaking changes.

Account Sync and Dashboard APIs

  • Added a signed-in AfroPoints account snapshot endpoint that returns profile, recent activity, recent contributions, and cashout history in one Supabase-backed response.
  • Updated the main dashboard AfroPoints lane to use the new account snapshot instead of duplicate profile/activity/cashout requests.
  • Improved profile and AfroPoints endpoints so bearer tokens and secure browser-session cookies follow the same auth path.
  • Marked account-bearing API responses as private, no-store and varied them by origin, authorization, and cookie to reduce stale cross-device account state.

AfroPoints

  • AfroPoints cockpit, cashout, submission, and contributor setup pages now clear stale account caches after profile, contribution, or payout actions.
  • Cashout views now prefer the Supabase account snapshot before falling back to narrower legacy calls.
  • Contributor payout details remain durable in Supabase and are reused by cashout flows when the saved method matches the request.
  • Dashboard and AfroPoints pages now dispatch and listen for account-change events so open tabs refresh faster after user actions.

AfroAlerts

  • Confirmed the live AfroAlerts flow: scheduled change detection writes significant price and rate changes to Supabase, sends high-severity email summaries when configured, and serves dashboard/admin alert views through /api/alerts.
  • Admin alert responses are now private and uncached, while the public active-alert feed keeps its short public cache for fast dashboard reads.

AI Advisor

  • Reduced the shared Anthropic input budget guard so AfroTools AI Advisor requests stay under the standard Claude 200K-token input ceiling.
  • No AfroTools AI helper request path uses the retired Anthropic 1M-context beta header.
Migration impact. No client migration is required. Existing dashboard, profile, alert, and AfroPoints routes continue to work under the same URLs.

v1.0 March 2026

Initial public release of the AfroTools API platform with five API products covering all 54 African countries.

AfroTax API

  • Income tax (PAYE) calculations for all 54 African countries
  • Corporate tax and withholding tax support
  • Capital gains tax calculations
  • Gross-to-net and net-to-gross calculation modes
  • Pension and social security deductions
  • Monthly, annual, weekly, and biweekly pay periods
  • Country listing and detailed country info endpoints

AfroFX API

  • Real-time exchange rates for 42 African currencies
  • Bid/ask spreads from central bank sources
  • Currency conversion with custom amounts
  • 24-hour change tracking with percentage and absolute values
  • Historical rates available on Pro plan and above

AfroFuel API

  • Fuel prices for all 54 African countries
  • Petrol (PMS), diesel (AGO), kerosene (DPK), and LPG coverage
  • Weekly updates from national petroleum regulators
  • Price change tracking since last update
  • Optional currency conversion for cross-country comparison

AfroRates API

  • Central bank policy rates for all 54 countries
  • Headline CPI inflation data
  • Treasury bill yields (91-day, 182-day, 364-day)
  • Prime lending and deposit rates
  • MPC meeting date tracking (last and next)
  • Historical data available on Pro plan and above

AfroVAT API

  • VAT/GST calculation for all 54 African countries
  • Add mode (add VAT to net amount) and extract mode (extract VAT from gross amount)
  • Standard, reduced, and zero rate support
  • Category-based rate lookup for food, medical, education, digital, and exports

Platform

  • API key authentication with afro_test_ and afro_live_ key modes
  • Four pricing tiers: Free ($0), Starter ($29/mo), Pro ($99/mo), Enterprise ($299/mo)
  • Rate limiting with daily and monthly quotas and informative response headers
  • Official SDKs for JavaScript/Node.js and Python
  • Two-panel API documentation with code examples in 5 languages
  • Interactive API Explorer for testing endpoints in the browser

Versioning Policy
Breaking changes are released under a new API version (e.g., v2). Deprecated versions receive at least 90 days of continued support. Non-breaking additions (new fields, new endpoints) are added to the current version without notice.
