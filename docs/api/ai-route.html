POST /v1/ai/route

AI Route API

Send a short user task and receive a validated AfroTools tool or workflow link. The endpoint is designed for partner sites, widgets, schools, job boards, publishers, and internal portals that need safe routing without exposing internal prompts or unrestricted model behavior.

Request

Base URL: https://afrotools.com/api/v1. Use an API key in the x-api-key header. Test keys (afro_test_*) return deterministic sandbox data with separate sandbox limits.

FieldTypeRequiredDescription
querystringYesUser task to route, up to 1,200 characters.
countrystringNoCountry hint used for route prefill only. Do not send private user data.
localestringNoUI language hint such as en, fr, pt, ar, or sw.
allowedCategoriesarray or comma stringNoRestrict recommendations to categories such as career, education, business, trade, energy, documents, local-life, construction, or country-intelligence.
partnerContextobjectNoSmall non-sensitive context, for example {"surface":"jobs-board"}. Credential, CV, document, payment, contact, or identity fields are rejected.
curl -X POST https://afrotools.com/api/v1/ai/route \ -H "x-api-key: afro_test_partner_demo" \ -H "Content-Type: application/json" \ -d '{ "query": "Write me a CV for an electrical engineer in Ghana", "country": "Ghana", "allowedCategories": ["career", "business"], "partnerContext": { "surface": "jobs-board" } }'

Response

The endpoint returns a compact routing decision. It does not return internal prompts, chain-of-thought, raw provider payloads, raw query text, or unvalidated external URLs.

{ "status": "success", "selectedTool": { "id": "cv-builder", "name": "CV Builder", "href": "/tools/cv-builder/" }, "selectedRoute": "/tools/cv-builder/?source=api_ai_route&country=Ghana", "category": "cv-jobs", "confidence": 0.74, "missingInputs": [], "suggestedDisplayText": "Open CV Builder for this AfroTools workflow.", "privacyMode": "browser_local", "safetyDomain": "employment" }
FieldDescription
statussuccess, no_match, or error.
selectedToolValidated AfroTools tool id, display name, and public route base.
selectedRouteRoot-relative AfroTools route. External URLs and model-suggested routes are not returned.
categoryMatched workflow category or search for safe fallback.
confidenceDeterministic route confidence from 0 to 1.
missingInputsField names your UI can ask before opening a calculator or workflow.
suggestedDisplayTextShort partner-safe copy for a button or card.
privacyModeHow the underlying workflow treats user data, such as browser_local or ai_optional.
safetyDomainHigh-stakes domain label such as tax, finance, employment, education, energy, or none.

Sandbox Examples

Education

I want to study in Canada from Nigeria with a budget of $8,000 routes to the study abroad workflow and usually asks for study level.

Trade

How much duty will I pay to import a 2016 Toyota Axio into Nigeria? routes to import duty and asks for item value.

Energy

Should I install solar for my shop in Lagos? routes to solar ROI and asks for monthly power or fuel spend.

Safety Rules

  • API keys are required and rate limited. See Rate Limits.
  • The endpoint uses deterministic routing first and validates every returned tool id and route against the AfroTools manifest.
  • Partner context must be non-sensitive. Do not send raw CVs, documents, customer contact data, credentials, payment data, or identity numbers.
  • High-stakes domains are labeled so partner UIs can show the right warnings before users rely on tax, customs, immigration, finance, energy, or employment guidance.
