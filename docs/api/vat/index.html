AfroVAT API

VAT and GST calculation engine for African countries. Calculate tax-inclusive or tax-exclusive prices with category-based exemptions.

Overview

The AfroVAT API handles value-added tax calculations for e-commerce, invoicing, and accounting software. It understands country-specific VAT rates, reduced rates for essential goods, exempt categories, and zero-rated items.

Calculation Modes

ModeDescription
exclusiveGiven a net amount, add VAT on top. Returns gross amount and VAT.
inclusiveGiven a gross amount, extract the VAT included. Returns net amount and VAT.

Category-Based Rates

Many African countries apply different VAT rates based on the type of goods or services. The AfroVAT API supports category codes to automatically apply the correct rate:

  • standard — Standard VAT rate (default)
  • food_basic — Basic food items (often zero-rated or reduced)
  • medical — Medical supplies and pharmaceuticals
  • education — Educational materials and services
  • digital — Digital services and software
  • export — Exported goods (zero-rated)

Endpoints

POST Calculate VAT
Calculate VAT for a given amount, country, and category. Supports exclusive and inclusive modes.
Base URL
https://afrotools.com/api/vat
Quick Example
# Calculate VAT on 10,000 NGN (exclusive) curl -X POST https://afrotools.com/api/vat \ -H "x-api-key: afro_test_abc123" \ -H "Content-Type: application/json" \ -d '{ "country": "NG", "amount": 10000, "mode": "exclusive" }'
Example Response
{ "success": true, "data": { "country": "NG", "currency": "NGN", "vat_rate": 7.5, "category": "standard", "mode": "exclusive", "net_amount": 10000.00, "vat_amount": 750.00, "gross_amount": 10750.00 } }
