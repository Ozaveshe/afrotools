Calculate VAT

Calculate VAT for a given amount, country, and category with exclusive or inclusive modes.

POST https://afrotools.com/api/vat
Authentication required. Pass your API key via x-api-key header. Learn more

Request Body

ParameterTypeDescription
country required string ISO 3166-1 alpha-2 country code.
amount required number Monetary amount in local currency.
mode optional string exclusive (add VAT, default) or inclusive (extract VAT).
category optional string Product category: standard, food_basic, medical, education, digital, export. Default: standard.
items optional array Array of line items for multi-item invoices. Each item has amount, category, and optional description.

Response Fields

FieldTypeDescription
countrystringISO country code.
currencystringCurrency code.
vat_ratenumberApplied VAT rate (%).
categorystringCategory used for rate lookup.
modestringCalculation mode used.
net_amountnumberAmount before VAT.
vat_amountnumberVAT amount.
gross_amountnumberAmount including VAT.
itemsarrayBreakdown per item (if items were provided).

Error Responses

StatusCodeDescription
400invalid_countryCountry code not supported.
400invalid_amountAmount must be a positive number.
400invalid_categoryCategory is not recognized.
401missing_api_keyNo API key was provided.
429rate_limit_exceededRate limit exceeded.
Request Examples
curl -X POST https://afrotools.com/api/vat \ -H "x-api-key: afro_live_your_key_here" \ -H "Content-Type: application/json" \ -d '{ "country": "NG", "amount": 10000, "mode": "exclusive", "category": "standard" }'
const res = await fetch("https://afrotools.com/api/vat", { method: "POST", headers: { "x-api-key": "afro_live_your_key_here", "Content-Type": "application/json" }, body: JSON.stringify({ country: "NG", amount: 10000, mode: "exclusive", category: "standard" }) }); const data = await res.json(); console.log(`VAT: ${data.data.vat_amount}`);
