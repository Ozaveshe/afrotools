# AfroTools 100 Super Apps Directory

Last updated: 2026-06-11

This is the working record for the 100 apps AfroTools should treat as flagship products for product effort, advertising, partner conversations, and YouTube content. It extends the earlier Super Apps directory from 12 candidates into a full 100-app operating record.

## Program Shape

- Slots 1-80 are public or public-workspace apps that should pull traffic, trust, backlinks, sponsors, API interest, or email capture.
- Slots 81-90 are the 10 Pro control/compliance apps.
- Slots 91-100 are the 10 Pro Daily OS apps. They belong in the product universe, but should not be marketed as fully connected SaaS until their account-backed data layers are real.
- The public homepage should stay useful and discovery-led, with a stronger focus rail for the flagship apps.

## Super App Standard

- Complete workflow for a real user problem, not only a calculator shell.
- Clear source, freshness, privacy, or assumption notes where facts can drift.
- Mobile-first interaction with no horizontal overflow.
- Export, copy, save, share, packet, or handoff output where useful.
- Analytics that avoid raw sensitive content.
- Honest labels for local-only, account-backed, estimate-only, source-backed, shell, or blocked states.
- Passing targeted checks plus `npm run audit`, `npm run check-links`, and the relevant browser/workflow smoke before ad spend or YouTube traffic is pointed at the route.
