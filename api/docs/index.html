Authentication

All API requests require an API key passed in the Authorization header.

Authorization: Bearer YOUR_API_KEY

Get your API key from the AfroTools Dashboard. Free tier includes 100 API calls per day.

Keep your API key secret. Do not expose it in client-side code. Use server-side calls or environment variables.

Base URL

All API endpoints use the following base URL:

https://api.afrotools.com/v1

Error Codes

The API uses standard HTTP status codes. Errors return a JSON body with details.

{
  "error": {
    "code": "rate_limit_exceeded",
    "message": "You have exceeded your daily API call limit.",
    "status": 429
  }
}
StatusCodeDescription
400invalid_requestMissing or invalid parameters
401unauthorizedInvalid or missing API key
403forbiddenAPI key does not have access to this endpoint
404not_foundEndpoint or resource not found
429rate_limit_exceededDaily or per-minute rate limit exceeded
500internal_errorServer error. Try again or contact support.

PAYE Calculator

GET /api/v1/tax/paye

Calculate Pay-As-You-Earn tax for an employee in a specific African country.

Parameters

ParameterTypeRequiredDescription
countrystringRequiredISO 3166-1 alpha-2 country code (e.g., NG, ZA, KE)
salarynumberRequiredAnnual gross salary in local currency
periodstringOptionalPay period: "monthly", "annual" (default: "annual")

Response

{
  "country": "NG",
  "currency": "NGN",
  "gross_salary": 500000,
  "period": "annual",
  "tax_breakdown": {
    "taxable_income": 420000,
    "paye_tax": 42600,
    "pension": 40000,
    "nhf": 12500,
    "total_deductions": 95100,
    "net_salary": 404900
  },
  "effective_rate": 8.52,
  "brackets": [
    {"range": "0 - 300,000", "rate": 7, "tax": 21000},
    {"range": "300,001 - 600,000", "rate": 11, "tax": 13200}
  ]
}

VAT Rates

GET /api/v1/vat/rates

Get current VAT/GST rates for African countries.

Parameters

ParameterTypeRequiredDescription
countrystringOptionalISO 3166-1 alpha-2 code. Omit for all countries.

Response

{
  "rates": [
    {"country": "NG", "name": "Nigeria", "vat_rate": 7.5, "currency": "NGN"},
    {"country": "ZA", "name": "South Africa", "vat_rate": 15, "currency": "ZAR"},
    {"country": "KE", "name": "Kenya", "vat_rate": 16, "currency": "KES"}
  ],
  "updated_at": "2026-03-15T00:00:00Z"
}

Exchange Rates

GET /api/v1/fx/rates

Get real-time exchange rates for African currencies. Updated every 15 minutes.

Parameters

ParameterTypeRequiredDescription
basestringOptionalBase currency code (default: USD)
symbolsstringOptionalComma-separated currency codes (e.g., NGN,ZAR,KES)

Response

{
  "base": "USD",
  "timestamp": "2026-03-15T12:30:00Z",
  "rates": {
    "NGN": 1580.50,
    "ZAR": 18.25,
    "KES": 152.80,
    "GHS": 15.40,
    "EGP": 50.75
  }
}
