Tafuta na badilisha maandishi kwenye PDF

Zana ya AfroTools Tafuta na Badilisha PDF hukuruhusu kutafuta maandishi ndani ya PDF na kuyabadilisha kwa visual overlay. Inaunga mkono case-sensitive search, whole-word matching, regex, page ranges, selected-match replacement, style controls, preview navigation, undo na CSV audit export.

Uchakataji wote ni client-side. PDF yako haitoki kwenye browser. Zana hutumia pdf.js assets za ndani kwa text extraction na rendering, pamoja na pdf-lib runtime ya ndani kwa visual overlay.

Maswali ya mara kwa mara

Is my PDF uploaded to a server?
Hapana. Uchakataji wote hufanyika kwenye browser yako. PDF yako haitoki kwenye kifaa chako. Zana hutumia pdf.js kwa rendering na pdf-lib kwa modifications, zote client-side.
Visual replacement hufanya kazi vipi?
Zana huchora cover rectangle juu ya maandishi ya awali na kisha kuchora maandishi mapya juu yake. Hii ni visual overlay, si text edit halisi. Maandishi ya awali yanaweza bado kupatikana kwenye data layer ya PDF.
Naweza kutumia hii kuficha taarifa nyeti?
Zana hii hufanya visual replacement pekee. Maandishi ya awali yanaweza bado kuwepo kwenye data layer ya PDF. Kwa redaction ya kweli inayofuta maandishi kabisa, tumia zana maalum ya PDF redaction.
Does find and replace support page ranges and regex?
Yes. By default the search is case-insensitive across all pages. You can enable case-sensitive matching, whole-word matching, regex search, and custom page ranges such as 1-3, 5, 8.
What font is used for the replacement text?
Replacement text is rendered in Helvetica, a standard PDF font. The tool can auto-fit the font size to the covered text box or use a custom point size, with configurable text color, cover color, padding, and opacity.
