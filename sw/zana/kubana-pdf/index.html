Kubana PDF: How It Works

The AfroTools Kubana PDF reduces PDF file size directly in your browser. Clean mode performs a lossless rewrite that preserves selectable text where possible. Wastani, Kubwa, Juu quality, and Custom modes are built for scanned or image-heavy PDFs: they render pages at controlled DPI and JPEG quality, then rebuild a smaller PDF. The tool supports batch compression, ZIP download, optional grayscale conversion, target size attempts, page caps for very large documents, and before-after size reports. Matokeo depend on the source file: scanned documents usually shrink much more than already-optimized or mostly text PDFs. Your documents are never uploaded to a server.

Frequently Asked Questions

How much can my PDF be compressed?

Banaion results vary depending on the content. PDFs with high-resolution images can see 50-80% reduction. Maandishi-heavy PDFs with few images may see smaller reductions since text is already compact.

What do the quality presets mean?

Clean preserves document structure where possible. Wastani is a good default for sharing. Kubwa targets upload portals and email limits. Juu quality keeps more visual detail. Custom lets you set DPI, JPEG quality, grayscale, page cap, and optional target size.

Will compression make the text blurry?

Clean mode is the safest choice when selectable text matters. Wastani, Kubwa, Juu quality, and Custom modes rasterize each page, so text remains visually readable but may no longer be selectable. Use those modes mainly for scans and image-heavy files.

Is my PDF uploaded to a server?

No. The entire compression process runs in your browser using JavaScript. Your PDF never leaves your device, making it completely safe for confidential, legal, or sensitive documents.

Is there a limit on PDF file size?

There is no strict server limit since processing is client-side. However, very large PDFs (over 100MB) may take longer to process depending on your device's memory and processing power.
