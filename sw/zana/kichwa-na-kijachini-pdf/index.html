What This Covers
  1. Six zones for independent left, center, and right placement.
  2. Tokens for page numbers, totals, dates, filenames, and Bates IDs.
  3. Ukurasa targeting with ranges, odd/even filters, and start numbers.
  4. Batch output as a ZIP when more than one PDF is selected.

Professional PDF headers and footers in the browser

Use headers and footers when you need visible document control: page numbers, dates, confidentiality labels, file names, version notes, or Bates-style references. AfroTools keeps the workflow local and gives every position its own field so formal packs do not need manual page-by-page editing.

The export preserves the original PDF content and adds new text marks on top. Keep the original source file if you need a clean version later.

Precise placement

Set header and footer content independently across left, center, and right zones.

Document tokens

Insert page numbers, total pages, dates, filenames, titles, and Bates-style labels.

Batch ready

Apply one configuration to several PDFs and download the results as one ZIP file.
