    How this stays useful

    These presets are tuned for real jobs: seller photos, proof sheets, event coverage, and internal drafts. If the preset gets you close, the manual fields help you finish the job without starting over.

    • Use tiled marks for preview-only work that should not be cropped easily.
    • Use corner marks for seller photos where you want the item to stay visible.
    • Include a handle or phone line when the image itself is part of your sales funnel.

    Private by default

    Your batch is processed entirely in the browser with HTML canvas. That makes it safer for customer photos, unpublished work, event galleries, and internal drafts.
