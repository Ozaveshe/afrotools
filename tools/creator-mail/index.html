FAQ

Common questions

Can I send emails directly from CreatorMail?

No. CreatorMail is a design tool. You build your newsletter here, export the HTML, then paste it into your email platform (Mailchimp, ConvertKit, Brevo, etc.) to send.

Will my newsletter look good in all email clients?

Yes. The exported HTML uses table-based layouts with all CSS inlined — the gold standard for email compatibility. Works in Gmail, Outlook, Apple Mail, Yahoo, and mobile apps.

Can I use my own images?

Yes. Paste any image URL into the image block settings. Images are referenced by URL so they need to be hosted somewhere accessible.

Does it support merge tags?

Yes. Insert {{first_name}}, {{subscriber_count}}, and {{date}}. Tags are highlighted in the editor and preserved in exported HTML.

What export formats are available?

Copy email-ready HTML, download as .html file, or copy a plain-text version. HTML export has all CSS inlined with table layouts.

Are the templates free?

Yes. All 12 templates, unlimited builds, unlimited exports. No signup, no watermarks.
