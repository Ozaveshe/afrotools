Local editing and preview

Primary task

  • Load one or more PDFs and choose header or footer positions.
  • Insert page, date, filename, title, or Bates-style tokens where needed.
  • Preview placement, then download a PDF or ZIP batch output.

Result and export

  • Updated PDF with visible headers and footers.
  • Batch ZIP when multiple PDFs are selected.
  • Preview panel for checking margins and page targeting.

Before you use it

  • Confirm page ranges, odd/even rules, and start numbers.
  • Review overlays on scanned documents and narrow margins.
  • Use a dedicated redaction/signing workflow for sensitive edits beyond visible labels.

Assumptions and freshness

Professional PDF headers and footers in the browser

Use headers and footers when you need visible document control: page numbers, dates, confidentiality labels, file names, version notes, or Bates-style references. AfroTools keeps the workflow local and gives every position its own field so formal packs do not need manual page-by-page editing.

The export preserves the original PDF content and adds new text marks on top. Keep the original source file if you need a clean version later.
