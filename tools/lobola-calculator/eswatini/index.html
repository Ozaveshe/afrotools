Questions

Is lilobolo in Eswatini an official fixed rate?

No. Lilobolo expectations vary by family, community, and agreement. This page is a planning worksheet and should not be treated as an official rate.

Why does the Eswatini page use SZL defaults?

The page starts in lilangeni so families can budget locally while still editing livestock, gifts, cash, and ceremony assumptions.

How should I use the estimate with family?

Use the estimate as a conversation summary. Confirm each line directly with family representatives and avoid treating any sample value as settled.
