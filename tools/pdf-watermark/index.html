Watermarking runs locally and creates a browser-generated PDF copy. Watermarks are visual deterrents, not access control, copyright enforcement, or redaction.

Local-first PDF workflow

Reviewed 2026. Disclaimer: verify with the recipient before relying on watermarked output. The shared PDF account gate does not upload your files.

Methodology

  • Applies text or image overlays to selected pages using local PDF generation routines.
  • Opacity, rotation, placement, and repeat settings affect readability and file size.
  • Image watermarks work best with transparent PNG files prepared before upload.

Before sharing

  • Open the output in another reader and confirm page coverage, contrast, placement, file size, and print appearance.
  • Do not rely on watermarks for confidential redaction or document access control.
  • For signatures, page order, or visual covers, continue in PDF Workspace.
