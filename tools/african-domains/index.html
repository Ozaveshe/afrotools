💡 African Domain Quick Guide
🇳🇬
.ng / .com.ng
No local presence required for .ng. .com.ng is cheaper (~$10/yr). Registered via NiRA-accredited registrars like QServers, WhoGoHost.
🇰🇪
.ke / .co.ke
.co.ke is most popular ($15-20/yr). No local company required. Register via KENIC-accredited registrars like Truehost, Webmaster Kenya.
🇿🇦
.za / .co.za
.co.za requires registrant to be in SA or have SA company. $12-18/yr. Register via ZARC-accredited registrars: Afrihost, 1-grid, Hetzner SA.
🇬🇭
.gh / .com.gh
No local presence required for .com.gh. ~$25/yr. Register via NIC Ghana. Alternatives: .net.gh, .org.gh, .edu.gh (for institutions).
