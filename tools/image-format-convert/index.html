    Format reminders

    JPG is still the safest default for portals and marketplaces. WebP is usually best for websites. PNG is right for logos, screenshots, and anything that needs transparency.

    • Use JPG for forms, profile photos, and most upload portals.
    • Use WebP for web delivery when you want a smaller file with good visual quality.
    • Use PNG only when crisp text or transparency matters more than file size.
