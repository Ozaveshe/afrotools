Free online PDF editor for private document work

The AfroTools PDF Workspace runs core PDF editing tools in your browser. Organize pages, merge PDFs, split or extract pages, add text, images, highlights, signatures, dates, stamps, watermarks and visual cover boxes, then compress or export a checked copy. Account features such as Vault saving may require sign-in, but source files are not uploaded by default.

Available Tools

  • Merge PDF — Combine two or more PDF files into a single document. Drag to reorder pages before merging.
  • Split PDF — Extract specific pages or split a large document into smaller files by page range.
  • Compress PDF — Reduce PDF file size while keeping readable quality, ideal for email attachments.
  • Add Watermark — Overlay text or image watermarks on every page of your document.
  • Page Numbers — Add page numbers in various positions and formats to any PDF.
  • Protect PDF — Create a password-protected copy when the local encryption runtime is available.
  • Sign PDF — Draw or type your signature and place it on any page of a document.
  • Visual Cover — Add cover boxes, flatten the review copy where possible, and verify the exported file before sharing sensitive documents.

Privacy & Security

Core processing happens locally in your browser using JavaScript. AfroTools does not upload source PDFs, generated files, passwords, OCR text, or document contents unless a future workflow clearly asks and you choose it. Always keep an untouched original and verify the exported copy in another PDF reader before sending it.

Best fit

Use this workspace for everyday PDF jobs: combining office packets, removing extra pages, signing a form, adding page numbers, watermarking a draft, compressing an email attachment, or making a visual review copy. For certified e-signatures, legal redaction, official filing, or regulated records, use the recipient's required system and treat this as a preparation workspace.

Frequently Asked Questions

Is there a file size limit?

There is no fixed server limit for browser-local editing, but your device memory is the practical limit. Large scanned PDFs and many-page files may work better on a laptop than on an older phone.

Are my files uploaded to a server?

No. All PDF operations run locally in your browser using client-side JavaScript. Your documents never leave your device, ensuring complete privacy.

Does this work on mobile devices?

Yes. The upload screen and editor use touch-sized controls on phones and tablets. Complex operations such as merging many PDFs, OCR-heavy work, or large scanned documents are still more comfortable on desktop.

Workspace limits

Use the workspace as a local editing desk, then verify the exported copy

PDF Workspace organizes, annotates, signs, protects, compresses, and exports documents in this browser. It is not cloud storage, certified e-signature, OCR guarantee, or legal redaction software.

Local-first PDF workspace

Reviewed 2026. Disclaimer: verify with the recipient before relying on the exported copy. Local workspace metadata must not store source PDF bytes or document contents.

Methodology

  • Uses browser PDF libraries to load pages, track overlays, organize pages, and generate a new exported PDF copy.
  • Signatures, stamps, text, highlights, page order, and visual covers are flattened into the exported file where supported.
  • Saved recipes and report trails should remain metadata-only and local unless the signed-in workspace explicitly syncs metadata.

Before sharing

  • Open the export in another reader and check page order, overlays, signatures, visual covers, forms, links, file size, and permissions.
  • For sensitive redaction, do not rely on visual covers alone; verify the flattened output and use a specialist workflow when required.
  • For a repeatable route, start from the Document & PDF planner.
