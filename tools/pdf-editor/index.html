Free Online PDF Editor

The AfroTools PDF Editor lets you annotate and modify PDF documents directly in your browser. Add text anywhere on a page, insert images with drag-and-resize controls, draw rectangles, circles, and lines, highlight important passages, place signatures, stamp review status, white out old content, or sketch freehand notes. Every edit happens client-side using JavaScript, so your files never leave your device.

Editing Features

  • Text Tool — Click anywhere to place text with custom font size, color, and bold styling.
  • Image Insert — Upload PNG, JPG, or WebP images, then drag and resize them on any page.
  • Shape Drawing — Draw rectangles, circles, and lines with configurable stroke and fill colors.
  • Highlighter — Drag to highlight text passages with a semi-transparent overlay.
  • Freehand Draw — Natural pen strokes with smooth bezier interpolation.
  • Signature — Place a typed signature or use the drawing/image tools for hand signatures.
  • Stamp — Add Approved, Reviewed, Confidential, Draft, Paid, or Void labels.
  • Whiteout — Cover existing content before replacing it with new text.
  • Undo / Redo — Full history with Ctrl+Z and Ctrl+Y keyboard shortcuts.

Privacy & Security

All processing happens locally in your browser. Your files are never uploaded to any server, keeping sensitive documents completely private.

How to Use

  1. Upload your PDF — Drag and drop a file or click to browse. The file stays in your browser.
  2. Select a tool — Choose from the toolbar: Text, Image, Shape, Highlight, Draw, Whiteout, Sign, or Stamp.
  3. Edit the page — Click or drag on the PDF page to add your content. Use the properties bar to adjust colors, sizes, and styles.
  4. Navigate pages — Click thumbnails in the left sidebar or use arrow keys to move between pages.
  5. Undo mistakes — Press Ctrl+Z to undo or Ctrl+Y to redo. The full edit history is preserved.
  6. Download — Click the Download button to save your edited PDF with all changes baked in.

Supported File Types

The PDF Editor accepts standard PDF files. Very large documents depend on your device memory because rendering and export happen locally. For image insertion, PNG, JPEG, and WebP formats are supported. WebP images are converted to PNG during export for PDF reader compatibility.

Frequently Asked Questions

Can I edit the existing text in a PDF?

This tool adds new content on top of the PDF. To modify existing text, you would need to use the eraser tool to cover the original text and then add new text in its place.

What image formats can I insert?

You can insert PNG, JPEG, and WebP images. They will be embedded directly into the PDF when you download the edited file.

Can I sign and stamp a PDF?

Yes. Use Sign for a typed signature, Draw or Image for a hand signature, and Stamp for common workflow labels such as Approved, Reviewed, Confidential, Draft, Paid, or Void.

Is there a file size limit?

There is no hard limit. Processing happens in your browser, so it depends on your device's memory. Most devices handle files up to 100MB comfortably.

Are my files uploaded to a server?

No. Everything runs locally in your browser. Your documents never leave your device.

Does it work on mobile?

Yes. The editor is fully responsive with touch support for drawing and a mobile-optimized toolbar at the bottom of the screen.

Can I undo my changes?

Yes. Use Ctrl+Z to undo and Ctrl+Y to redo. The full edit history is preserved until you close the editor or download the file.

How do I add shapes with no fill?

Select the Shape tool, then click the "No Fill" button in the properties bar. This will draw only the outline of the shape with the stroke color you have selected.

Does the editor modify the original PDF content?

No. All edits are overlaid on top of the existing PDF content. The original text, images, and formatting remain untouched underneath your additions.
