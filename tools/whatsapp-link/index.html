Frequently Asked Questions

What is a wa.me link?
A wa.me link is an official WhatsApp short URL that opens a chat with a specific phone number. It works on mobile and desktop without needing to save the contact first.
Do I need to include the country code?
Yes. WhatsApp requires the full international phone number including country code (without the + or leading zero). Our tool handles the formatting automatically when you select a country.
Can I add a pre-filled message?
Yes. You can add a default message that will appear pre-typed in the chat. The recipient can still edit or clear it before sending. This is great for customer support or order inquiries.
What is the QR code for?
The QR code encodes your WhatsApp link. Print it on flyers, business cards, or display it in your shop so customers can scan and start a chat instantly.
How does bulk mode work?
Enter multiple phone numbers (one per line) and a shared message. The tool generates individual wa.me links for each number, which you can copy all at once or one by one.
