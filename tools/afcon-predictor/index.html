How it works
The simulator uses ELO-style ratings based on FIFA rankings, AFCON history, and recent results. Each match is simulated using win probability calculated from the rating difference. Group stages run round-robin; knockouts use single-elimination. Run 5,000+ times for stable probability estimates.
