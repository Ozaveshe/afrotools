Safe workflow

Portal labels can change. Read the current screen and KRA guidance before submitting.

1

For a new PIN, confirm the correct registration route

KRA says individual PIN registration is completed online on iTax. Requirements differ for residents, non-citizens, non-residents and special cases.

  • Start from KRA’s current individual registration guide.
  • Check the official requirements list for your status.
  • Do not upload identity documents to AfroTools.
2

For login or password recovery, stay on the official domain

Open the KRA portal directly. KRA’s PIN FAQ says password recovery begins from the iTax login page and uses the registered email address.

  • Check the address begins with https://itax.kra.go.ke/.
  • Never enter your KRA PIN, password or recovery code on this page.
  • Use KRA support if your registered email or account details are wrong.
3

Before filing, reconcile the return with source records

Gather the records relevant to every active tax obligation and income source. For employment income, compare the employer record with your own payroll documents before transferring any figure.

  • Confirm the taxpayer, obligation and return period shown by iTax.
  • Declare all taxable income categories that apply; do not assume the employer’s PAYE return replaces your individual return.
  • Use the current portal form and instructions—not an old screenshot or downloaded template.
  • Review the declaration, submit only on iTax and save the official acknowledgement.
4

Use NIL only when the facts and obligation support it

KRA’s filing FAQ describes the NIL option inside the Returns menu for periods without income. A NIL submission is not a shortcut for missing records or unreported income.

  • Check which obligation and period are active in your profile.
  • If you had income or are unsure, stop and use KRA or a qualified tax adviser.
  • Keep the official acknowledgement after submission.
5

Use account history for previous submissions

KRA’s FAQ says previous returns can be viewed through the Returns area using the consult-return option, return type and period. Use the receipt from iTax as the record—not an AfroTools checklist.

Common questions

Does AfroTools submit anything to KRA?

No. This page has no login, PIN field, upload, filing integration or payment connection.

Does every PIN holder file the same return?

No. Obligations and circumstances differ. Inspect the active obligations in your iTax profile and current KRA guidance.

Can this page tell me to file NIL?

No. It explains KRA’s NIL route but cannot know whether you had income or which obligation applies.

Where should I save proof?

Save the acknowledgement or receipt generated by the official KRA system after a successful action.

Before opening iTax

  • Know the exact taxpayer and task.
  • Have your own official records ready.
  • Check the active obligation and period.
  • Use a private device and trusted connection.
  • Plan where to store the official receipt.
Security boundary

This guide never asks for a KRA PIN, password, OTP, ID scan, P9 or return file. Do not paste them into any AfroTools field.
