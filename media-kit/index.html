FAQ

Short answers for buyers before a call.

Can a sponsor change calculator logic?

No. Sponsors can fund placement, distribution, or a custom build, but they cannot buy changes to independent formulas, source notes, or outputs.

How fast can a pilot launch?

A simple sponsored page or widget pilot can usually launch in 3 to 5 business days after assets, target pages, attribution, and compliance notes are confirmed.

Can AfroTools support white-label widgets?

Yes. White-label work is a paid pilot with brand, attribution, analytics, lead routing, and maintenance terms defined before launch.

Can sponsors receive leads?

Yes, when the user explicitly submits a form. AfroTools should not sell hidden widget usage data as sponsor leads.

Do you share traffic numbers?

Share current analytics privately during sales calls. Public pages should not invent reach claims that are not backed by live data.

Where should technical buyers go?

API and data buyers should review the API overview, API pricing, and developer resources before scoping a subscription.
