Launch lanes

Pick the lane before you touch files. Each one tells Codex what to read, what to change, and what to run.

Pages

Page and registry work

Read the tool or country guide, touch the source page, then validate links and registry behavior.

  • Use docs/ADDING-A-TOOL.md or docs/ADDING-A-COUNTRY.md
  • Update registry only if discovery changed
  • Run link and audit checks
npm run check-links
SEO

SEO operations

Route metadata, internal links, aliases, and OG behavior through existing scripts before mass manual edits.

  • Use afrotools-seo-ops
  • Treat sitemap files as generated
  • Validate with SEO-aware scripts
npm run seo:report
QA

Release QA

Map touched files to the right validation surface instead of always reaching for the biggest command.

  • Use afrotools-release-qa
  • Choose narrow checks first
  • Call out residual risk clearly
npm test
Live

Supabase-aware changes

Separate live project questions from repo edits and verify live assumptions before coding around them.

  • Use the configured Supabase MCP server first
  • Keep live and repo actions distinct
  • Capture new conventions once proven
Use configured supabase MCP first
Pro

Gate coverage

Use the generated Pro gate coverage artifact before changing Pro routes, settings, success, or app pages.

  • Source: admin/data/pro-gate-coverage.json
  • Regenerate with the audit script
  • Public Pro page stays indexable
node scripts/audit-pro-gate-coverage.js

House style

These rules are now centralized in the design doctrine instead of hiding across CSS, HTML, and standards docs.

Open doctrine
Rule

Use the system

Start from design-system.css tokens and only add custom CSS for composition or special cases.

Rule

Typography carries the brand

Instrument Serif is for the statement, DM Sans for the work surface, JetBrains Mono for commands.

Rule

Blue drives, gold accents

Lean on the repo’s blue family for action and state. Use gold only for emphasis or signal.

Rule

Composition before cards

Prefer rhythm, scale, layout, and contrast before turning every region into another box.

Rule

Utility copy for operations

Internal surfaces should orient the operator, not sound like a campaign page.

Rule

Motion with restraint

Use reveals and hover emphasis only when they improve hierarchy and still respect reduced motion.
