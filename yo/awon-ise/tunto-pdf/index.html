Bí a ṣe ń lò ó
1. Gbe PDF wá nipa fífa-sílẹ̀ tàbí yíyan fáìlì
2. Fa ojú-iwe láti tún ìtò wọn ṣe ninu grid
3. Tẹ ojú-iwe, lo Shift+tẹ, tàbí apoti àárín láti yan ojú-iwe
4. Yí, ṣe ẹ̀dà, yí ìtò padà, yọ jade, fi kun, tàbí pa ojú-iwe tí a yan rẹ́
5. Tẹ "Gba PDF silẹ̀" láti gba ìwé tuntun
6. Lo "Dá padà" fun iṣẹ́ tó kẹhin tàbí "Tún bẹ̀rẹ̀" láti pada sí PDF tí o gbe wá

Ọna iṣẹ ati asiri

PDF rẹ duro ninu aṣàwákiri rẹ.

Irinṣẹ yii n ka oju-iwe PDF, n ṣe awọn aworan kekere, o si n kọ faili tuntun lori ẹrọ rẹ. A ko gbe iwe rẹ si server AfroTools. Ti PDF ba tobi ju tabi ba ni aabo ọrọigbaniwọle, aṣàwákiri le nilo akoko diẹ sii tabi o le nilo lati pin iṣẹ naa si awọn faili kekere.

Awọn irinṣẹ ti a lo

  • pdf-lib fun kikọ PDF tuntun.
  • Mozilla PDF.js fun kika ati wiwo oju-iwe.

Túntò ojú-iwe PDF: bí ó ṣe ń ṣiṣẹ́

Irinṣẹ́ AfroTools fún títún ojú-iwe PDF tò ń jẹ́ kí o ṣakoso ojú-iwe ninu PDF rẹ. Gbe PDF wá, wo ojú-iwe kọọkan bí àwòrán kékeré, fa wọn sí ìtò tuntun, yí scan tó yí sẹ́gbẹ̀ẹ́, pa ojú-iwe tí o kò fẹ́ rẹ́, ṣe ẹ̀dà, yí ìtò padà, fi ojú-iwe lati PDF míì kun, tàbí yọ ojú-iwe tí a yan sí fáìlì tuntun. Àárín bíi 1-3, 7, 10-12 ràn ọ́ lọwọ fun ìwé gígùn. Irinṣẹ́ yìí ń dá PDF tuntun ninu aṣàwákiri rẹ, fáìlì sì dúró lori ẹrọ rẹ.

Àwọn ìbéèrè tí a máa ń bi

Báwo ni mo ṣe tún ojú-iwe tò?

Lẹ́yìn tí o bá gbe PDF wá, o máa rí àwòrán kékeré fun ojú-iwe kọọkan. Fa ojú-iwe sí ibi tuntun, ìtò náà yóò yí loju iboju.

Ṣé mo lè yí ojú-iwe kọọkan?

Bẹẹni. Ojú-iwe kọọkan ní bọtini yíyí fún ìyí 90. Ó wúlò fun scan tó yí sẹ́gbẹ̀ẹ́ tàbí sókè-sílẹ̀.

Ṣé mo lè pa ojú-iwe pàtó rẹ́?

Bẹẹni. Tẹ bọtini Pa rẹ́ lori ojú-iwe tí o fẹ́ yọ. Àwọn tó kù yóò tún nọ́mbà ara wọn ṣe. Gba PDF tuntun silẹ̀ nígbà tí o bá tẹ́lọ́run.

Ṣé mo lè fi ojú-iwe lati PDF míì kun?

Bẹẹni. Lo Fi ojú-iwe kun láti ṣafikun PDF kan tàbí diẹ̀ sii. Tí ojú-iwe kan ṣoṣo bá wà ní yíyan, a fi tuntun sí lẹ́yìn rẹ; bí bẹ́ẹ̀ kọ́, a fi sí ipari.

Ṣé mo lè yọ ojú-iwe tí mo yan sí PDF tuntun?

Bẹẹni. Yan ojú-iwe pẹlu ọwọ́ tàbí àárín, lẹ́yìn náà tẹ Yọ jade. Ojú-iwe tí a yan yóò gba silẹ̀ bí PDF lọtọ.

Ṣé títún ìtò ṣe máa dín didara PDF ku?

Rárá. Irinṣẹ́ yìí tún ojú-iwe tò lai compress tàbí yí akoonu wọn padà. Ọ̀rọ̀, àwòrán ati ìtò dúró bí wọ́n ṣe wà ninu PDF atilẹba.

Ṣé a máa gbe ìwé mi sí server?

Rárá. Gbogbo títún ojú-iwe ṣe ń ṣẹlẹ̀ ninu aṣàwákiri rẹ. PDF rẹ kò kúrò lori ẹrọ rẹ.
